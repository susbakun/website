# راهنمای سریع شروع

## 🚀 نصب و اجرا (5 دقیقه)

### 1. نصب Backend
```bash
cd backend
npm install
npm run seed    # اضافه کردن محصولات نمونه (اختیاری)
npm run dev
```

✅ Backend روی پورت **5001** اجرا می‌شود

### 2. نصب Frontend
```bash
cd frontend
npm install
npm run dev
```

✅ Frontend روی پورت **3000** یا **3001** اجرا می‌شود

## 🎯 دسترسی به سیستم

### مشتری عادی
می‌توانید ثبت‌نام کنید و خرید کنید

### ادمین پیش‌فرض
```
Email: admin@shop.com
Password: admin123
```

**دسترسی‌های ادمین:**
- مدیریت محصولات
- تأیید اصالت محصولات
- مدیریت سفارشات
- مدیریت کاربران

## 📦 دیتابیس

پروژه از **SQLite** استفاده می‌کند که:
- نیازی به نصب جداگانه ندارد
- فایل دیتابیس: `backend/database.sqlite`
- به صورت خودکار ایجاد می‌شود
- ادمین پیش‌فرض خودکار ساخته می‌شود

## 🔗 لینک‌های مهم

- **Frontend**: http://localhost:3001
- **Backend API**: http://localhost:5001
- **پنل ادمین**: http://localhost:3001/admin

## ✨ ویژگی‌های اصلی

1. **تأیید اصالت کالا** - کارشناسان محصولات را تأیید می‌کنند
2. **ارسال رایگان** - برای خریدهای بالای 500 هزار تومان
3. **بازگشت 30 روزه** - ضمانت بازگشت وجه
4. **کد رهگیری** - پیگیری آنلاین سفارش
5. **پنل مدیریت** - مدیریت کامل محصولات، سفارشات و کاربران

## 🛠️ تست سریع

### 1. ورود به پنل ادمین
1. برو به http://localhost:3001/login
2. وارد کن: `admin@shop.com` / `admin123`
3. کلیک روی "پنل مدیریت" در منو

### 2. افزودن محصول
1. پنل مدیریت → مدیریت محصولات
2. کلیک "+ افزودن محصول جدید"
3. پر کردن فرم و ذخیره

### 3. تأیید محصول
1. در لیست محصولات، کلیک "تأیید"
2. نوشتن یادداشت کارشناسی
3. تأیید محصول

### 4. خرید به عنوان مشتری
1. خروج از حساب ادمین
2. ثبت‌نام با ایمیل جدید
3. مشاهده محصولات تأیید شده
4. افزودن به سبد و خرید

## 📝 نکات مهم

- **فایل دیتابیس** در `backend/database.sqlite` ذخیره می‌شود
- برای **reset کردن دیتابیس**، فقط این فایل را حذف کنید
- **ادمین پیش‌فرض** هر بار که سرور start می‌شود، بررسی و در صورت نبود ایجاد می‌شود
- **محصولات تأیید نشده** در صفحه اصلی نمایش داده نمی‌شوند

## 🐛 رفع مشکلات

### پورت در حال استفاده است
```bash
# Kill کردن پروسس روی پورت 5001
lsof -ti:5001 | xargs kill -9

# Kill کردن پروسس روی پورت 3000
lsof -ti:3000 | xargs kill -9
```

### دیتابیس خراب شده
```bash
# حذف فایل دیتابیس
rm backend/database.sqlite

# Restart کردن backend
cd backend
npm run dev
```

### خطای npm
```bash
# پاک کردن node_modules و نصب مجدد
rm -rf node_modules package-lock.json
npm install
```
